# Responsive Calculator (HTML • CSS • Vanilla JavaScript)

## Features
- Modern dark theme UI with glassmorphism.
- Calculator display with **real-time result preview** as you type.
- Buttons: digits (0–9), decimal (`.`), percentage (`%`), clear (`AC`), delete (`⌫`), operators (`+ − × ÷`), equals (`=`).
- **Keyboard support**:
  - Numbers: `0-9`
  - Operators: `+ - * /`
  - Equals: `Enter` or `=`
  - Delete: `Backspace`
  - Clear: `Esc` or `Delete`
  - Percentage: `%`
- Error handling: **division by zero / invalid expressions** show `Error`.
- Fully responsive layout (works on desktop + mobile).

## Technologies Used
- HTML5
- CSS3 (modern layout + glassmorphism styling)
- Vanilla JavaScript (no frameworks)

## How to Run
1. Open `index.html` in any modern browser.
2. Use mouse/touch or the keyboard shortcuts listed above.

## Keyboard Shortcuts
- `0-9` : append digit
- `.` : decimal
- `+  -  *  /` : operators
- `%` : percent (current number becomes `n / 100`)
- `Enter` or `=` : evaluate
- `Backspace` : delete last input
- `Esc` or `Delete` : clear (AC)
